# OpenIA Generator

Technologies

- **[Vite](https://vitejs.dev/)**
- **[Tailwind](https://tailwindcss.com/)**
- **[Vue 3](https://vuejs.org/)**

Run

- _npm install_
- _npm run dev_

And then

- Clone _.env.example_ and rename as _.env_
- Change **[API KEY](https://platform.openai.com/account/api-keys)**
